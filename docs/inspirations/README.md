# Inspirations Folder

This folder contains reference images for design inspiration.

## How to Use

1. **Add images** - Drop screenshots, designs, or mood boards into this folder
2. **Use clear naming** - Name files descriptively so they're easy to reference

## Recommended Naming Convention

```
01-overall-style.png      # Overall design direction
02-color-scheme.png       # Color palette reference
03-typography.png         # Font/text styling reference
04-navigation.png         # Navigation design
05-hero-section.png       # Hero/header design
06-cards.png              # Card component styles
07-buttons.png            # Button styles
08-forms.png              # Form/input styles
09-footer.png             # Footer design
10-mobile-view.png        # Mobile responsive reference
```

## Supported Formats

- `.png` (preferred for screenshots)
- `.jpg` / `.jpeg`
- `.webp`
- `.gif` (for animation references)
- `.svg` (for icon/illustration references)

## Organizing by Category

For larger projects, create subfolders:

```
/inspirations
├── /colors          # Color palette references
├── /typography      # Font and text style references
├── /layouts         # Page layout references
├── /components      # UI component references
├── /animations      # Animation/motion references
└── /mood-boards     # Overall aesthetic references
```

## After Adding Images

1. Open `design-inspirations.md` in your project
2. Document each image with what elements to match
3. Specify priority (High/Medium/Low) for each reference
4. Note specific elements you want to incorporate

## Tips

- **Be specific** - Don't just add images, note exactly what to match
- **Prioritize** - Mark which references are most important
- **Mix sources** - Combine elements from multiple references
- **Capture details** - Take close-up screenshots of specific components you like
- **Include context** - Add notes about why you chose each reference
